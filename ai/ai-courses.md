# AI学习课程整理

以下是一份整理的AI相关学习课程列表，涵盖从入门到进阶、从基础模型到行业应用的各种资源。这些课程主要来自Coursera、edX、Udacity、Fast.ai，以及Stanford、MIT、CMU、Berkeley等高校公开课程。建议根据自己的基础和目标方向选择课程，不必一次性全部学习。

## 学习路径建议
- **零基础入门**: Python基础 -> 线性代数/概率统计 -> 机器学习入门 -> 小项目实践。
- **算法与模型方向**: 机器学习 -> 深度学习 -> NLP/CV/推荐系统 -> 论文复现与模型优化。
- **大模型应用方向**: Python/后端基础 -> LLM API -> RAG -> Agent -> 评测与部署。
- **工程落地方向**: 数据工程 -> 模型训练 -> MLOps -> 云平台部署 -> 监控与迭代。
- **行业结合方向**: 先补目标行业知识，再学习AI在该行业的数据、模型、合规和业务指标。

## 入门级课程
- **Coursera: 机器学习 (Machine Learning)** - Andrew Ng  
  经典入门课程，涵盖监督学习、无监督学习等基础概念。  
  链接: [Coursera](https://www.coursera.org/learn/machine-learning)

- **edX: 人工智能导论 (Introduction to Artificial Intelligence)** - MIT  
  介绍AI的基本概念、历史和应用。  
  链接: [edX](https://www.edx.org/course/introduction-to-artificial-intelligence)

- **Udacity: 人工智能入门 (Intro to Artificial Intelligence)**  
  免费课程，适合初学者了解AI基础。  
  链接: [Udacity](https://www.udacity.com/course/intro-to-artificial-intelligence--cs271)

- **DeepLearning.AI: AI For Everyone** - Andrew Ng  
  面向非算法背景学习者，帮助理解AI能做什么、不能做什么，以及如何判断AI项目价值。  
  链接: [Coursera](https://www.coursera.org/learn/ai-for-everyone)

- **Kaggle Learn: Intro to Machine Learning**  
  通过短教程和Notebook练习快速上手机器学习流程，适合作为实践入门。  
  链接: [Kaggle Learn](https://www.kaggle.com/learn/intro-to-machine-learning)

- **Google: Introduction to Generative AI**  
  介绍生成式AI、基础模型和常见应用场景，适合了解大模型时代的基础概念。  
  链接: [Google Cloud Skills Boost](https://www.cloudskillsboost.google/course_templates/536)

## 方向选择建议
- **想快速转AI应用开发**: 优先学Prompt Engineering、LLM API、RAG、Agent、向量数据库和部署，适合前端、后端、全栈工程师。
- **想走算法岗或研究岗**: 优先学数学基础、机器学习、深度学习、NLP/CV、论文阅读与复现，周期更长但基础更扎实。
- **想做企业AI落地**: 优先学MLOps、数据工程、模型评测、权限安全、成本优化和业务指标设计。
- **想结合现有行业经验**: 选择医疗、金融、教育、制造、安全等垂直方向，把行业数据和业务问题作为核心竞争力。
- **想做创业或独立产品**: 优先从AI工具、自动化工作流、知识库问答、内容生成、客服/销售助手等低门槛场景切入。
