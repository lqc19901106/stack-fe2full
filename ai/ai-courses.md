# AI学习课程整理

以下是一份整理的AI相关学习课程列表，涵盖从入门到进阶、从基础模型到行业应用的各种资源。这些课程主要来自Coursera、edX、Udacity、Fast.ai，以及Stanford、MIT、CMU、Berkeley等高校公开课程。建议根据自己的基础和目标方向选择课程，不必一次性全部学习。

## 学习路径建议
- **零基础入门**: Python基础 -> 线性代数/概率统计 -> 机器学习入门 -> 小项目实践。
- **算法与模型方向**: 机器学习 -> 深度学习 -> NLP/CV/推荐系统 -> 论文复现与模型优化。
- **大模型应用方向**: Python/后端基础 -> LLM API -> RAG -> Agent -> 评测与部署。
- **工程落地方向**: 数据工程 -> 模型训练 -> MLOps -> 云平台部署 -> 监控与迭代。
- **行业结合方向**: 先补目标行业知识，再学习AI在该行业的数据、模型、合规和业务指标。

### 工程控制方向能力模块
- **控制理论基础**: 动态系统、状态空间、稳定性、可控性/可观性、PID、频域分析。
- **状态估计与系统辨识**: Kalman Filter、EKF/UKF、参数辨识、灰盒建模、数据驱动建模。
- **最优控制与MPC**: LQR/LQG、轨迹优化、约束优化、模型预测控制、实时求解器。
- **强化学习控制**: MDP、策略梯度、Actor-Critic、SAC/PPO、模型型强化学习、仿真到现实迁移。
- **安全与鲁棒控制**: 约束满足、鲁棒性、故障安全、控制屏障函数、异常工况降级策略。
- **智能控制工程化**: Simulink、Python控制库、acados、实时仿真、边缘部署、控制器监控。
- **数字孪生与仿真环境**: 物理建模、仿真数据生成、策略评估、硬件在环/软件在环测试。
- **行业控制场景**: 机器人、无人车、飞控、过程工业、电机控制、暖通空调、储能和智能制造。

### 工程控制实践项目建议
- **倒立摆智能控制**: 分别实现PID、LQR、MPC和强化学习控制，对比稳定性、收敛速度和鲁棒性。
- **移动机器人轨迹跟踪**: 基于MPC或Pure Pursuit实现路径跟踪，并加入障碍物约束和速度限制。
- **电机速度控制系统**: 建立电机模型，比较PID、状态反馈、MPC和数据驱动控制效果。
- **强化学习连续控制Demo**: 使用Gymnasium/MuJoCo训练SAC/PPO控制器，完成摆杆、车杆或机械臂任务。
- **过程控制优化器**: 针对温度、压力、流量等过程变量建立预测模型并做MPC控制。
- **控制系统异常检测**: 使用传感器数据识别执行器故障、传感器漂移和模型失配。
- **数字孪生控制台**: 构建仿真环境，实时展示状态、控制量、约束、预测轨迹和告警。
- **安全强化学习策略评估**: 给RL策略加入动作边界、状态约束和安全过滤器，比较约束前后表现。

- **SOH与剩余寿命预测**: 使用NASA或公开电池循环数据集，预测容量衰减曲线和剩余可用循环次数。
- **电池异常检测系统**: 对电芯电压差、温度梯度、内阻变化和采样异常做无监督异常检测与告警分级。

- **储能电池调度系统**: 结合电价、负荷、光伏出力和电池寿命成本，优化储能充放电计划。
- **BMS知识库助手**: 基于电池规范、告警码、维修手册和历史故障案例构建RAG问答与故障诊断助手。
